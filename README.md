# HBD Countdown

Project Next.js sederhana untuk halaman hitung mundur menuju 9 Juli 2026.

## Jalankan Lokal

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Deploy ke Vercel

Push folder ini ke repository Git, lalu import repository ke Vercel. Tidak ada backend atau API route di project ini.
